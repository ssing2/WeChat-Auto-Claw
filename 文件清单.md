**格式**：

**规则**：
- 每行一个场景
- 空行忽略
- # 开头为注释

---

### 5. publisher.py

**用途**：发布器核心脚本

**核心类**：
```python
class WeChatPublisher:
    - get_access_token()      # 获取Token
    - get_material_images()   # 获取素材库图片
    - render_with_wenyan()    # 文颜渲染
    - publish_single()        # 发布单篇
    - publish_multi_group()   # 发布多图文
```

**使用示例**：
```python
from scripts.publisher import WeChatPublisher

publisher = WeChatPublisher(app_id, app_secret)
publisher.publish_multi_group(
    markdown_files=["a.md", "b.md", "c.md", "d.md", "e.md"],
    theme="orangeheart",
    random_cover=True
)
```

### 6. batch_publish_multi.py

**用途**：批量发布脚本

**功能**：
- 读取场景列表
- 生成文章
- 渲染排版
- 发布多图文

**使用示例**：
```bash
python scripts/batch_publish_multi.py \
  --config config/config.yaml \
  --scenarios config/scenarios.txt
```
