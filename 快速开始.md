# 微信公众号自动发布系统 - 快速开始指南

## 🚀 5分钟快速上手

### 第1步：安装环境（2分钟）

```bash
# 安装Python依赖
pip install requests beautifulsoup4

# 安装文颜CLI
npm install -g @wenyan-md/cli

# 验证安装
python --version  # Python 3.8+
npx @wenyan-md/cli --version
```

---

### 第2步：配置系统（2分钟）

创建配置文件 `config/config.yaml`：

```yaml
wechat:
  app_id: "你的微信AppID"
  app_secret: "你的微信AppSecret"

article:
  author: "你的作者名"
  
publish:
  articles_per_group: 5
  use_random_cover: true
```

**获取AppID和AppSecret**：
1. 登录 https://mp.weixin.qq.com
2. 设置 → 开发 → 基本配置
3. 复制AppID和AppSecret

**配置IP白名单**：
1. 同上页面 → IP白名单
2. 添加服务器IP地址

---

### 第3步：生成并发布（1分钟）

```bash
# 创建场景文件
echo "入职不签合同怎么办" > scenarios.txt

# 生成并发布
python scripts/batch_publish_multi.py \
  --scenarios scenarios.txt \
  --group-size 5 \
  --theme orangeheart
```

## 🎯 三种使用方式

### 方式1：命令行（推荐新手）

#### 发布单篇

```bash
python scripts/publish_single.py \
  --file article.md \
  --theme orangeheart
```

#### 批量发布多图文

```bash
python scripts/batch_publish_multi.py \
  --scenarios scenarios.txt \
  --group-size 5
```

---

### 方式2：Python脚本（推荐开发者）

```python
from scripts.publisher import WeChatPublisher

# 初始化
publisher = WeChatPublisher(
    app_id="你的AppID",
    app_secret="你的AppSecret"
)

# 发布单篇
publisher.publish_single(
    markdown_file="article.md",
    theme="orangeheart"
)

# 发布多图文
publisher.publish_multi_group(
    markdown_files=["a.md", "b.md", "c.md", "d.md", "e.md"],
    theme="orangeheart",
    random_cover=True
)
```

---

### 方式3：配置文件（推荐批量）

创建 `batch_config.yaml`：

```yaml
input:
  scenarios_file: "scenarios.txt"
  articles_dir: "articles/to-publish"

output:
  published_dir: "articles/published"
  log_file: "logs/publish.log"

publish:
  group_size: 5
  theme: orangeheart
  random_cover: true
  
wechat:
  app_id: "xxx"
  app_secret: "xxx"
```

运行：

```bash
python scripts/batch_publish_multi.py \
  --config batch_config.yaml
```

## ⚠️ 重要提示

### 1. IP白名单配置

**必须配置**，否则无法发布！

```
登录公众号 → 设置 → 开发 → 基本配置 → IP白名单
添加服务器IP地址
```

### 2. 标题长度限制

- **限制**：64字节（UTF-8编码）
- **中文**：约20-30个汉字
- **自动截断**：系统会自动截断超长标题

### 3. 封面图要求

- **尺寸**：900x383px（推荐）
- **格式**：JPG/PNG
- **大小**：不超过5MB
- **必须**：先上传到素材库获取media_id

### 4. 铁律

**所有文章只发草稿箱，老板审阅后正式发布！**

## 🆘 遇到问题？

### 常见错误

#### 错误1：IP不在白名单

```
解决方案：
1. 登录微信公众平台
2. 设置 → 开发 → 基本配置
3. IP白名单 → 添加服务器IP
```

#### 错误2：access_token无效

```
解决方案：
1. 检查AppID和AppSecret是否正确
2. 删除cache/token.json
3. 重新获取token
```

#### 错误3：中文乱码

```
解决方案：
确保使用UTF-8编码：
export PYTHONIOENCODING=utf-8
```
