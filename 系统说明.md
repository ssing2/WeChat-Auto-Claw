老板，**微信公众号自动发布系统**已经整理完成！

### 📦 系统包含

#### 核心功能

- ✅ **自动文章生成**：基于场景稿生成专业法律科普文章

## 📋 使用说明

### 3步快速开始

#### 第1步：配置

编辑 `config/config.yaml`：

```yaml
wechat:
  app_id: "你的微信AppID"        # 改为你的AppID
  app_secret: "你的微信AppSecret"  # 改为你的AppSecret

article:
  author: "你的作者名"            # 改为你的作者名
```

#### 第2步：配置IP白名单

1. 登录 https://mp.weixin.qq.com
2. 设置 → 开发 → 基本配置
3. IP白名单 → 添加服务器IP

#### 第3步：运行

```bash
python scripts/batch_publish_multi.py \
  --config config/config.yaml \
  --scenarios config/scenarios.txt
```

## 🔧 核心脚本说明

### publisher.py

**功能**：发布到微信公众号

**核心类**：
- `WeChatPublisher` - 发布器类

**核心方法**：
- `publish_single()` - 发布单篇
- `publish_multi_group()` - 发布多图文
- `render_with_wenyan()` - 文颜渲染
- `get_material_images()` - 获取素材库图片

### batch_publish_multi.py

**功能**：批量发布多图文

**参数**：
- `--config` - 配置文件
- `--scenarios` - 场景稿文件
- `--group-size` - 每组文章数
- `--theme` - 文颜主题

## 📝 配置说明

### config.yaml

```yaml
# 微信公众号配置
wechat:
  app_id: "xxx"                    # 你的AppID
  app_secret: "xxx"                # 你的AppSecret

# 文章配置
article:
  author: "袁律"                    # 作者名
  min_words: 800                    # 最小字数
  max_words: 1500                   # 最大字数

# 文颜配置
wenyan:
  theme: "orangeheart"              # 主题
  highlight: "solarized-light"      # 代码高亮

# 发布配置
publish:
  articles_per_group: 5             # 每组文章数
  use_random_cover: true            # 随机封面
```

## 🚀 使用示例

### 发布单篇

```bash
python scripts/publisher.py \
  --file article.md \
  --theme orangeheart
```

### 发布多图文

```bash
python scripts/batch_publish_multi.py \
  --scenarios config/scenarios.txt \
  --group-size 5 \
  --theme orangeheart
```

### Python脚本调用

```python
from scripts.publisher import WeChatPublisher

publisher = WeChatPublisher(
    app_id="xxx",
    app_secret="xxx"
)

publisher.publish_multi_group(
    markdown_files=["a.md", "b.md", "c.md", "d.md", "e.md"],
    theme="orangeheart",
    random_cover=True
)
```

## ⚠️ 重要提示

### 1. IP白名单

**必须配置**，否则无法发布！

```
公众号后台 → 设置 → 开发 → 基本配置 → IP白名单
```

### 2. 标题长度

- **限制**：64字节（UTF-8）
- **中文**：约20-30个汉字
- **自动截断**：系统自动处理

### 3. 封面图

- **尺寸**：900x383px（推荐）
- **格式**：JPG/PNG
- **大小**：不超过5MB
- **必须**：先上传到素材库

### 4. 铁律

**所有文章只发草稿箱，老板审阅后正式发布！**
