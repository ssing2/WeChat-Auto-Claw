# 微信公众号自动发布系统 - 安装部署指南

## 📦 系统要求

### 硬件要求

- **CPU**: 1核及以上
- **内存**: 2GB及以上
- **磁盘**: 10GB及以上
- **网络**: 稳定的互联网连接

### 软件要求

- **操作系统**: Linux / macOS / Windows
- **Python**: 3.8 或更高版本
- **Node.js**: 16.0 或更高版本
- **npm**: 7.0 或更高版本（或yarn）

---

## 🚀 安装步骤

### 第1步：安装Python依赖

```bash
# 使用pip安装
pip install requests beautifulsoup4 pyyaml

# 或使用requirements.txt
pip install -r requirements.txt
```

**requirements.txt内容**：
```
requests>=2.28.0
beautifulsoup4>=4.11.0
pyyaml>=6.0
```

---

### 第2步：安装Node.js和npm

#### Linux（Ubuntu/Debian）

```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

#### macOS

```bash
brew install node
```

#### Windows

下载安装包：https://nodejs.org/

---

### 第3步：安装文颜CLI

```bash
# 全局安装
npm install -g @wenyan-md/cli

# 验证安装
npx @wenyan-md/cli --version
```

如果安装失败，使用npx：
```bash
npx @wenyan-md/cli publish --help
```

---

### 第4步：下载系统

```bash
# 方式1：Git克隆（推荐）
git clone https://github.com/your-repo/公众号自动发布系统.git
cd 公众号自动发布系统

# 方式2：下载ZIP
# 下载并解压到本地目录
```

---

### 第5步：配置系统

#### 5.1 复制配置文件

```bash
cd config
cp config.example.yaml config.yaml
```

#### 5.2 编辑config.yaml

```bash
vim config.yaml
# 或使用其他编辑器
nano config.yaml
```

**必须修改的配置**：

```yaml
wechat:
  app_id: "你的微信AppID"        # 改为你的AppID
  app_secret: "你的微信AppSecret"  # 改为你的AppSecret

article:
  author: "你的作者名"            # 改为你的作者名
```

#### 5.3 获取微信AppID和AppSecret

1. 登录 https://mp.weixin.qq.com
2. 设置 → 开发 → 基本配置
3. 复制AppID和AppSecret
4. 粘贴到config.yaml

---

### 第6步：配置IP白名单

#### 6.1 获取服务器IP

```bash
# 查看公网IP
curl ifconfig.me

# 或
curl ipinfo.io/ip
```

#### 6.2 添加到微信白名单

1. 登录 https://mp.weixin.qq.com
2. 设置 → 开发 → 基本配置
3. IP白名单 → 添加IP地址
4. 保存

---

### 第7步：创建必要目录

```bash
mkdir -p articles/generated
mkdir -p articles/to-publish
mkdir -p articles/published
mkdir -p cache
mkdir -p logs
```

---

### 第8步：验证安装

```bash
# 检查Python
python --version

# 检查Node.js
node --version

# 检查npm
npm --version

# 检查文颜
npx @wenyan-md/cli --version

# 检查配置
python scripts/check_config.py
```

## 🔧 常见安装问题

### 问题1：pip安装失败

**解决方案**：
```bash
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple requests beautifulsoup4 pyyaml
```

### 问题2：npm安装失败

**解决方案**：
```bash
# 清除缓存
npm cache clean --force

# 使用国内镜像
npm install -g @wenyan-md/cli --registry=https://registry.npmmirror.com
```

### 问题3：权限错误

**解决方案**：
```bash
# Linux/macOS
sudo pip install requests beautifulsoup4 pyyaml
sudo npm install -g @wenyan-md/cli

# 或使用用户安装
pip install --user requests beautifulsoup4 pyyaml
```

### 问题4：文颜命令不存在

**解决方案**：
```bash
# 使用npx运行
npx @wenyan-md/cli --version

# 或添加到PATH
export PATH="$PATH:$(npm config get prefix)/bin"
```

## 🚀 生产环境部署

### 使用systemd管理（Linux）

创建服务文件 `/etc/systemd/system/wechat-publisher.service`：

```ini
[Unit]
Description=微信公众号自动发布系统
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/公众号自动发布系统
ExecStart=/usr/bin/python3 scripts/batch_publish_multi.py --config config/config.yaml
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动服务：

```bash
sudo systemctl daemon-reload
sudo systemctl enable wechat-publisher
sudo systemctl start wechat-publisher
sudo systemctl status wechat-publisher
```

## 📊 性能优化

### 1. 使用虚拟环境

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
source venv/bin/activate  # Linux/macOS
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

### 2. 并发发布

修改 `config/config.yaml`：

```yaml
performance:
  max_workers: 4  # 增加并发数
```

### 3. 缓存Token

```yaml
wechat:
  access_token_cache: "cache/token.json"
  token_expire_buffer: 300  # 提前5分钟刷新
```
